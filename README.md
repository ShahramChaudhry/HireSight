# HireSight

An AI-powered recruitment platform that helps recruiters evaluate candidates using intelligent CV analysis and LinkedIn profile evaluation.

## Features

- **Dashboard**: View active job posts and candidate statistics
- **Upload CVs**: Bulk upload and process candidate resumes
- **Define Criteria**: Set custom evaluation criteria with weighted scoring
- **LinkedIn Integration**: Evaluate candidates via LinkedIn profiles
- **Candidate Rankings**: View and sort candidates by scores
- **Detailed Analysis**: AI-powered candidate evaluation with key highlights

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Lucide React** - Icons
